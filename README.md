# memeify

![demo image](demo.png)

memey scripts for imagemagick

    usage: memeify input output [args]
    -m / --caption "text" "text"  add caption
    -q / --cubify                 cubify it
    -d / --deep-fry               deep fry it
    -l / --liquid-rescale         liquid rescale it
    -i / --implode                implode it
    -e / --explode                explode it
    -s / --swirl                  swirl it
    -n / --invert                 invert it
